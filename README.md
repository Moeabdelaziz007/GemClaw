## 🚀 EXPANSION PROTOCOL (كيفية البدء)

1. **Manifest the Files:**
   ```bash
   git clone [repository-url]
   cd Gemigram
   ```

2. **Fuel the Core:**
   Create `.env` with:
   `NEXT_PUBLIC_GEMINI_API_KEY=your_key_here`

3. **Ignition:**
   ```bash
   npm install
   npm run dev
   ```
